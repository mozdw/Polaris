# Polaris
간단한 점프 게임입니다. 좌우 방향키로 이동, 위 방향키로 점프가 가능합니다. 발판을 밟고 최대한 높이 올라가 보세요.
중간중간 나오는 아이템을 먹으면 한 번 게임 오버를 막아주는 보호막이 생깁니다. 이미 보호막이 있다면 대신 아주 높게 점프를 합니다.
